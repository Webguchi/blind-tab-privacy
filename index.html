Privacy Policy — Blind Tab

Last updated: 2025-05-18

Data Collection

Blind Tab does not collect, transmit, or share any personal data. No information is sent to external servers.

Local Storage

The extension saves the following settings locally on your device using chrome.storage.local:

  • Selected mode (Spotlight / Selection / Hold)
  • Blur intensity
  • Spotlight radius

This data never leaves your device.

Permissions

  • storage — to save your settings locally
  • activeTab — to apply the overlay to the current tab only
  • scripting — to inject the overlay script into the current tab

Contact

For questions, please open an issue on GitHub.
